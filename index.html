What is Kinect?

Kinect is a device which senses motion developed by Microsoft originally for Xbox 360 and Xbox One video game consoles, which also supports Windows PCs. It is based around a webcam-style add-on peripheral which serves as a media of interaction between users and their console/computer, all without the need of any game controller, but rather with gestures and spoken commands.

Not only for playing games, Kinect also supports developers who are willing to use Kinect for other varieties of purposes, since Microsoft released a non-commercial Kinect software development kit (SDK) for Windows on February 21, 2011.

Developers are able to build applications with C++, C#, or Visual Basic using Microsoft Visual Studio and includes four main features: raw sensor streams, skeletal tracking, advanced audio capabilities, and sample code and documentation. Since then Kinect has been improved and developed to be the Kinect Version 2 that is available as the most advanced Kinect today.

Kinect Specification and Capabilities

The Kinect provides hardware capabilities to recognize human motion. This feature can be used as a tool of interaction in the virtual space and the augmented space. The Kinect device has an Infra-Red(IR) emitter, an IR depth sensor, and an RGB camera. These together provide the capabilities to calculate the human pose. The Kinect sensor range of skeletal tracking is from 1.2 to 3.5 meters. The sensor has 43 degrees vertical by 57 degrees horizontal field of view. It supports up to 30 FPS frame rate. The resolution of the depth stream is 320x240 in pixel units. The depth accuracy is about 5 cm at 5 m distance.

Kinect Skeletal Tracking


Skeletal tracking is the capability to track the skeleton image of one or two people moving within Kinect's field of view for gesture-driven applications, which would be the main ingredient for avatars to move according to user's gestures.

The Jedi in the game moves like you do: their hands swings as your hands swings, their head moves like yours. It feels like you, yourself, is in the game. Now that's what is called avatar animation.

Avatar Animation has been used as the basic of either post-production games (like Kinect Sports, Kinect Sports: Season Two, Nicktoons MLB, Dance Central 2, Just Dance 3, and whatnot) or currently-under-development games. It uses skeletons tracked with skeletal tracking, which then are encapsulated by skins, which resembles the character you are playing.
